# 个人品牌网站 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目名称
个人品牌网站 (Personal Brand Website)

### 1.2 项目背景
构建一个专业的个人品牌展示平台，用于展示个人经历、项目作品、技术博客，提升个人影响力和职业形象。

### 1.3 项目目标
- 打造简约高级的个人品牌形象
- 提供优质的内容展示体验
- 实现高效的内容管理系统
- 优化 SEO，提升搜索引擎可见度
- 支持多端响应式访问

### 1.4 目标用户
- **主要用户**：访客（潜在雇主、合作伙伴、技术爱好者）
- **管理员**：网站所有者（内容创作和管理）

---

## 2. 技术栈

### 2.1 前端技术
- **框架**：Next.js 15 (App Router)
- **语言**：TypeScript
- **样式**：TailwindCSS
- **组件库**：Shadcn UI
- **状态管理**：React Context / Zustand
- **表单验证**：React Hook Form + Zod
- **Markdown 渲染**：react-markdown + remark/rehype 插件

### 2.2 后端技术
- **BaaS**：Supabase
  - 认证系统
  - 数据库 (PostgreSQL)
  - 存储服务
  - 实时订阅
  - Row Level Security (RLS)

### 2.3 部署与运维
- **前端部署**：Vercel
- **数据库**：Supabase (托管 PostgreSQL)
- **CDN**：Vercel Edge Network
- **域名**：自定义域名 + SSL

### 3.4 用户系统 (User System)

#### 3.4.1 注册功能
- **注册方式**
  - 邮箱 + 密码注册
  - 第三方登录（Google / GitHub）

- **注册流程**
  1. 填写邮箱、密码、确认密码
  2. 邮箱验证（发送验证链接）
  3. 点击验证链接激活账号
  4. 跳转登录页

- **表单验证**
  - 邮箱格式验证
  - 密码强度验证（至少8位，包含字母和数字）
  - 密码一致性验证
  - 实时错误提示

#### 3.4.2 登录功能
- **登录方式**
  - 邮箱 + 密码登录
  - Google OAuth 登录
  - GitHub OAuth 登录

- **登录流程**
  1. 输入邮箱和密码
  2. 验证身份
  3. 登录成功，跳转到之前页面或首页

- **附加功能**
  - 记住我（7天免登录）
  - 忘记密码（邮箱重置）
  - 登录失败提示

#### 3.4.3 用户权限
- **普通用户**
  - 浏览所有公开内容
  - 点赞和收藏文章
  - 评论文章

- **管理员**
  - 所有普通用户权限
  - 访问管理后台
  - 发布/编辑/删除博客
  - 上传/编辑/删除项目
  - 查看数据统计

### 3.7 响应式设计

#### 3.7.1 断点设计
- **移动端**：< 640px
- **平板**：640px - 1024px
- **桌面端**：> 1024px

#### 3.7.2 适配要点
- **导航栏**
  - 桌面：横向导航
  - 移动：汉堡菜单

- **布局**
  - 桌面：多列布局
  - 移动：单列布局

- **字体大小**
  - 桌面：16px 基准
  - 移动：14px 基准

- **触摸优化**
  - 按钮最小尺寸 44x44px
  - 增加点击区域
  - 优化滚动体验

### 3.8 设计风格

#### 3.8.1 视觉风格
- **整体风格**：简约高级、黑白极简
- **参考**：Apple 官网、Vercel 官网
- **色彩方案**
  - 主色：黑色 (#000000)
  - 辅助色：白色 (#FFFFFF)
  - 灰色系：#F5F5F5, #E5E5E5, #999999
  - 强调色：蓝色 (#0070F3) 或自定义

#### 3.8.2 排版
- **字体**
  - 英文：Inter / SF Pro Display
  - 中文：思源黑体 / 苹方
  - 代码：JetBrains Mono / Fira Code

- **字号层级**
  - H1: 48px / 36px (桌面/移动)
  - H2: 36px / 28px
  - H3: 24px / 20px
  - Body: 16px / 14px
  - Small: 14px / 12px

#### 3.8.3 间距
- **基准单位**：8px
- **常用间距**：8px, 16px, 24px, 32px, 48px, 64px

#### 3.8.4 动效
- **过渡时间**：150ms - 300ms
- **缓动函数**：ease-in-out
- **常用动效**
  - 悬停状态变化
  - 页面切换淡入淡出
  - 滚动视差效果
  - 加载骨架屏

## 4. 非功能需求

### 4.1 性能要求
- **首屏加载时间**：< 2s
- **Lighthouse 分数**：> 90
- **Core Web Vitals**
  - LCP < 2.5s
  - FID < 100ms
  - CLS < 0.1

### 4.2 安全要求
- **认证安全**
  - JWT Token 认证
  - HTTPS 加密传输
  - XSS 防护
  - CSRF 防护

- **数据安全**
  - Row Level Security (RLS)
  - SQL 注入防护
  - 敏感信息加密

### 4.3 可用性要求
- **服务可用性**：99.9%
- **错误处理**：友好的错误提示
- **降级方案**：服务异常时的备用方案

### 4.4 可维护性
- **代码规范**：ESLint + Prettier
- **类型安全**：TypeScript 严格模式
- **文档完善**：README + 代码注释
- **版本控制**：Git + 语义化版本
